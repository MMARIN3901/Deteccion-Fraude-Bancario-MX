# 🛡️ Engine de Monitoreo Transaccional y Mitigación de Fraude Operativo
**Estratega de Datos:** Moisés Antonio Marín Bernal

## 📌 Resumen del Proyecto
Diseño y ejecución de un motor de auditoría de datos orientado a la banca comercial en México. El sistema automatiza la identificación de patrones delictivos mediante una arquitectura híbrida de Python y SQL.

## 🚀 Escenarios de Detección Implementados
* **Velocity Attacks:** Identificación de cargos múltiples en ventanas de tiempo críticas (< 5 min).
* **Location Anomalies:** Detección de transacciones inconsistentes con el perfil geográfico del usuario.
* **Exposure Monitoring:** Alertas automáticas por superación de umbrales financieros ($20,000+ MXN).

## 🛠️ Stack Tecnológico
* **Lenguajes:** Python 3, SQL (SQLite)
* **Librerías:** Pandas, Numpy
* **Entorno:** JupyterLab / Anaconda
